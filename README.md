# Signal Proxy

Running a Signal proxy for the people in Iran to connect and talk to each other again.
Freedom of Speech is important.


## Requirements
* terraform
* scaleway keys
* project id
* ssh-key
* ansible
* geerlingguy.docker

## Usage

```$ terraform apply```
```$ ansible-playbook site.yml```
